## 포함 범위

- Douzone SCM 영역을 기준으로 한 Copan SCM Process Asset 구축
- SCM TO-BE Overview와 Process Detail 관리
- SCM 관련 Capability Coverage 관리
- 공통 Master 정의
- Process Definition 해석 구조
- Layout Engine과 Routing Engine 안정화
- Property Panel과 Editor 생산성 개선
- Audit, Review, Decision 기반 변경 관리

## SCM 구축 대상 Capability

Navigator 구축 대상은 Douzone TO-BE 전체가 아니라 Douzone SCM 영역이다.

- 계약/프로젝트
- 구매관리
- 매입/입고관리
- 영업/주문관리
- 출고관리
- 재고관리
- 반품관리
- 정산관리
- 해외업무
- 플랫폼
- 콘텐츠/서비스
- 이벤트
- 공연장/팝업
- 매장/POS

위 Capability는 Navigator 안에서 Copan Process Asset으로 구축한다.

## 제외 범위

- 공통관리
- 조직관리
- 인사관리
- 회계관리
- 세무관리
- 무제한 자유 드로잉 도구
- 확정되지 않은 Generator 자동 생성
- Architecture 승인 없는 대규모 데이터 마이그레이션
- 현재 업무 기준과 무관한 시각 효과 중심 기능

## Reference Only 범위

공통관리, 조직관리, 인사관리, 회계관리, 세무관리는 Navigator 구축 대상이 아니다.

이 영역은 OmniEsol ERP 교육과 실제 사용으로 정착한다. Navigator에서는 SCM 프로세스와의 연결 지점, ERP 메뉴 참조, Coverage 제외 사유만 Reference Only로 관리한다.

## Coverage 기준

Coverage는 Douzone SCM TO-BE 기준으로 관리한다.

SCM Capability 100% 구축이 현재 프로젝트의 목표다.

Finance, HR, Tax, Common Administration 영역은 SCM Navigator Coverage 대상에서 제외하고 Reference Only로 관리한다.
