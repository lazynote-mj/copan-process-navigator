# Node Master

|Field|Value|
|---|---|
|Title|Node Master|
|Purpose|노드 유형, 기본 시스템, 기본 스타일, 기본 배치 참조 기준을 정의한다.|
|Status|Draft|
|Owner|Project Team|
|Last Updated|2026-06-27|
|Related Docs|`../01_Architecture/MasterModel.md`, `NodeDefinitionStandard.md`, `BusinessActivity.md`|

Node Master는 업무 활동이 화면에서 어떤 노드 유형으로 표현되는지 정의한다.

Node의 업무적 정의 기준은 `NodeDefinitionStandard.md`를 따른다.

기본 필드:
- nodeType
- label
- system
- defaultLane
- defaultPhase
- defaultSlot
- defaultStyle
- businessActivityId
