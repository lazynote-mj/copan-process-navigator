Process Pattern은 Generator가 반복 프로세스를 만들 때 참조할 수 있는 흐름 템플릿이다.

예:
- 주문 등록 → 출고 → 매출마감 → 전표생성
- 구매요청 → 발주 → 입고 → 매입마감
- 재고이동 요청 → 이동확정 → 재고 반영
